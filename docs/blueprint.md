# **App Name**: EstilosIA

## Core Features:

- Control de Temperatura Mejorado: Implementa un control deslizante para que los usuarios ingresen rangos de temperatura.
- Selección de Estilo Visual: Muestra las opciones de estilo usando tarjetas con iconos representativos para una selección visualmente intuitiva.
- Conexión a Base de Datos: Conéctate a la base de datos Supabase con la URL proporcionada y la clave anonpublic.
- Visualización de Atuendo Sugerido: Presenta las prendas sugeridas en tarjetas individuales, con imágenes o placeholders si no hay imagen disponible.
- Explicación Inteligente de Atuendo: Genera explicaciones de atuendos utilizando IA, which uses a tool to determine if it needs to incorporate armario del usuario information. Justifica la selección en base a temperatura y estilo, personalizando según los datos del usuario.

## Style Guidelines:

- Colores primarios: Tonos neutros y suaves (blanco, gris claro) para un aspecto limpio y moderno.
- Colores secundarios: Azul claro (#E1F5FE) y verde suave (#E8F5E9) para detalles y acentos.
- Acento: Un tono coral (#FF7043) para llamadas a la acción y elementos interactivos.
- Priorizar la legibilidad con textos claros y bien contrastados.
- Iconos minimalistas y modernos para la navegación y representación de categorías.
- Diseño limpio y espaciado generoso para una experiencia de usuario intuitiva.
- Transiciones suaves y sutiles al cambiar entre secciones y al interactuar con elementos.
